## 1. 리디자인 개요

### 1.1 이전 vs 이후 비교

#### **이전 (Generic)**
- 모든 사용자에게 동일한 정보 제공
- AI 이슈 지수: 통합만 제공
- 추천 알고리즘: 단순 랭킹
- 목표: "AI 정보를 쉽게 이해"

#### **이후 (Ainus - 맞춤형)**
- 🎯 직업/나이/성별에 따른 개인화
- 📊 통합 + 직업별 이슈 지수 이원화
- 🤖 작업 기반 AI 추천 (성능 기준)
- ⚠️ AI 범죄/위험 인사이트 추가
- 📈 AI 발전을 명확히 보여주기 (위기감 전달)
- 👥 커뮤니티 기반 지식 공유 및 참여

### 1.2 새로운 가치 제안

| 기능 | 사용자 이득 |
|------|-----------|
| **직업별 이슈 지수** | "내 일이 AI로부터 얼마나 영향받나?" 한눈에 파악 |
| **AI 위기감 시각화** | 과거 vs 현재 비교로 변화의 심각성 인식 |
| **인사이트 피드** | AI 범죄, 규제, 보안 등 실생활 관련 뉴스 (숏츠 형식) |
| **작업 기반 추천** | "이 일을 하려면?" → 최적 AI 즉시 제시 |
| **커뮤니티 참여** | 프롬프트 공유, 꿀팁 교환, 실시간 이슈 토론 |

- ⚠️ "내 직업도 영향받을 수 있다" 위기감 형성

#### 📰 **Priority 2: 인사이트 피드 (숏츠 형식)**

**목적**: AI 관련 실생활 이슈를 **숏츠 카드로 한눈에**

**포함 콘텐츠**:

| 카테고리 | 예시 | 중요도 |
|---------|------|--------|
| 🚀 **기업 발표** | Gemini 3 출시, Sora 2 공개 | ⭐⭐⭐⭐⭐ |
| 🛡️ **AI 범죄** | "딥페이크 사기 증가", "AI 성인물 불법 제작" | ⭐⭐⭐⭐⭐ |
| ⚖️ **규제/법안** | "EU AI 법안 강화", "한국 생성AI 규제안" | ⭐⭐⭐⭐ |
| 💼 **직무 영향** | "AI로 일자리 32% 감소 예상" | ⭐⭐⭐⭐ |
| 🔒 **보안/개인정보** | "ChatGPT 데이터 유출 사고" | ⭐⭐⭐⭐ |
| 💡 **활용 기술** | "의료 AI 진단 정확도 98% 달성" | ⭐⭐⭐ |
| 🎓 **교육/가이드** | "프롬프트 엔지니어링 완벽 가이드" | ⭐⭐ |

**카드 인터랙션**:
- 📱 **수직 스와이프 (위로)**: 다음 카드로 전환
- 👆 **카드 탭**: 상세 인사이트 페이지로 이동
- 👍 **하단 액션**: 도움됨, 댓글, 공유
- ⬅️➡️ **좌우 스와이프**: 이전/다음 카드 또는 필터 변경


```
앱 시작
  ↓

  │
  ├─ 🤖 AI 추천
  │   ├─ 작업 입력
  │   ├─ 추천 결과 (1~3위)
  │   └─ 상세 비교
  │
  ├─ 💬 커뮤니티
  │   ├─ 메인 (실시간 생성 게시판)

  │
  └─ 👤 마이페이지
      ├─ 프로필 정보
      ├─ 직업별 상세 분석
      ├─ 저장한 추천
      ├─ 내가 쓴 글
      └─ 알림 설정
```

### 5.2 이슈 지수 계산 로직

```python
def calculate_issue_index(job_category, date):
    """
    직업별 AI 이슈 지수 계산
    
    Parameters:
    - job_category: "소프트웨어 개발자"
    - date: "2025-01-16"
    
    Returns:
    - index_score: 0-100
    - main_issues: 주요 이슈 리스트
    - trend: 전일/전주 대비 변화
    """
    
    # 1단계: 전체 AI 이슈 수집
    all_issues = fetch_recent_ai_news(days=7)
    
    # 2단계: 직업별 관련도 필터링
    job_related_issues = filter_by_job_relevance(
        all_issues, 
        job_category
    )
    
    # 3단계: 이슈 중요도 계산
    scores = []
    for issue in job_related_issues:
        importance = calculate_importance(
            mention_count=issue['mentions'],
            source_credibility=issue['source_score'],
            impact_level=issue['impact'],
            job_relevance=issue['job_relevance']
        )
        scores.append(importance)
    
    # 4단계: 통합 지수 산출
    index_score = weighted_average(scores)
    
    # 5단계: 트렌드 계산
    previous_score = get_previous_index(job_category, date-1)
    trend = index_score - previous_score
    
    return {
        'index_score': index_score,
        'main_issues': job_related_issues[:3],
        'trend': trend,
        'trend_direction': 'up' if trend > 0 else 'down'
    }
```

```python
def recommend_ai_for_task(task_description, user_profile):
    """
    사용자가 입력한 작업에 최적 AI 추천
    
    Parameters:
    - task_description: "이미지에서 텍스트 추출"
    - user_profile: {직업, 예산, 속도, 정확도 우선순위}
    
    Returns:
    - ranked_ai_list: [1위, 2위, 3위, ...]
    - reasons: 각 순위 이유
    - benchmarks: 성능 지표 + 출처
    """
    
    # 1단계: 작업 카테고리 분류
    category = classify_task(task_description)  # "이미지 인식"
    
    # 2단계: 해당 카테고리 AI 모델 필터링
    candidates = filter_models_by_category(category)
    
    # 3단계: 성능 벤치마크 기반 순위 매김
    scores = calculate_performance_scores(
        candidates,
        metrics=['accuracy', 'speed', 'cost'],
        user_priority=user_profile['priority']  # e.g., 정확도 > 속도 > 비용
    )
    
    # 4단계: 사용자 맞춤 정렬
    ranked = rank_by_user_profile(scores, user_profile)
    
    # 5단계: 상세 정보 추가
    for ai in ranked:
        ai['benchmarks'] = get_benchmarks(ai['model_id'])
        ai['pros_cons'] = analyze_pros_cons(ai)
        ai['use_cases'] = get_use_cases(ai['model_id'])
    
    return {
        'ranked_list': ranked[:3],  # 상위 3개
        'comparison_table': create_comparison_table(ranked[:3]),
        'sources': get_data_sources(),
        'last_updated': get_last_update_time()
    }

def calculate_performance_scores(candidates, metrics, user_priority):
    """
    성능 점수 계산 (가중 평균)
    
    Returns:
    - scores: {model_id: weighted_score}
    """
    weights = {
        'accuracy': user_priority.get('accuracy', 0.4),
        'speed': user_priority.get('speed', 0.3),
        'cost': user_priority.get('cost', 0.3)
    }
    
    scores = {}
    for model in candidates:
        score = (
            model['accuracy'] * weights['accuracy'] +
            (1 / model['speed']) * weights['speed'] +  # 속도는 역수
            (1 / model['cost']) * weights['cost']  # 비용은 역수
        ) * 100
        scores[model['model_id']] = score
    
    return scores
```
